A hands-on **Java number guessing application** designed to help learners master core programming concepts. This project emphasizes problem-solving, user interaction, and incremental logic building while maintaining clean, readable, and extensible code.

---

## 📖 Learning Objectives
- Understand **Java syntax** and **data types**
- Practice **control flow** (loops, conditionals, switch)
- Apply **functions and methods**
- Implement **exception handling**
- Explore **file handling** and **database integration**
- Strengthen **object-oriented programming** concepts (encapsulation, constructors, constants)

---

## 🚀 Features & Use Cases

### 1. Game Initialization
- Random number generation within a predefined range
- Configurable maximum attempts and hint counter
- Welcome message and rules display
- **Key Concepts:** primitive data types, constructors, encapsulation, constants

---

### 2. User Guess Submission
- Accepts and validates user input
- Compares guess with target number
- Tracks attempts and provides feedback (*higher/lower/correct*)
- **Key Concepts:** Scanner input, control flow, loops, input validation

---

### 3. Hint Generation
- Provides hints after incorrect guesses (even/odd, divisibility, range)
- Limits number of hints to maintain challenge
- **Key Concepts:** conditional logic, arithmetic operators, method abstraction

---

### 4. Error Handling & Validation
- Handles non-numeric input gracefully
- Prevents out-of-range guesses
- Displays user-friendly error messages
- **Key Concepts:** exception handling, custom exceptions, input sanitization

---

### 5. Game Result Storage
- Stores results (player name, attempts, win/loss) in file or database
- Retrieves past game records
- Displays previous scores
- **Key Concepts:** File I/O, JDBC, object persistence, serialization

---

### 6. Game Restart & Exit
- Allows replay without restarting the program
- Resets game variables
- Closes resources properly
- **Key Concepts:** loop control, boolean flags, lifecycle handling
